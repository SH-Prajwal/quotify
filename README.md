# Quote Generator

A simple and interactive Quote Generator built using HTML, CSS, and JavaScript.  
It displays motivational quotes along with their authors and generates a new quote every time the user clicks a button.

## Features

- Generates random motivational quotes
- Displays quote author dynamically
- Simple and clean user interface

## API 

The project includes commented-out code for fetching quotes from an API.
To enable it:
- Add a valid API URL
- Uncomment the related function calls in `script.js`
